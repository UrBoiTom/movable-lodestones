At first glance, a mod that makes lodestones movable, and makes compasses forget them, might not seem super useful. But, thanks to some impressive mechanics discovered by [DanCraftHero](https://youtube.com/@dancrafthero) on YouTube, removing and replacing a lodestone can be used to send wireless redstone signals anywhere in your world. And, with this mod, you'll be able to do so literally with the press of a button.
